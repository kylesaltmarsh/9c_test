# 9c Test Calculator

Calculates your potential climbing grade based on the 9c test metrics.

## Usage

```bash
python 9c.py data/07_04_25.yaml
```

## Data Format

Create a YAML file with your test results:

```yaml
weight: 82
max_finger_strength: 45
max_pull_up: 75
max_core: l_sit_20
max_bar_hang: 150
```

Results are saved to `results/`.
